# 教育卡字符集清单（v8 启蒙版）

> 给美工 / 字体打包工具用。基于 v8 设计稿 + 975MaruSC 圆角字体。

## 字符集组成

| 段落 | 字符数 | Unicode 范围 | 用途 |
|---|---|---|---|
| ASCII 基础 | 95 | `0x20-0x7E` | 英文字母 + 数字 + 基础符号 |
| 拼音声调 | 24 | 拉丁补充 | `ā á ǎ à ē é ě è ī í ǐ ì ō ó ǒ ò ū ú ǔ ù ǖ ǘ ǚ ǜ ü` |
| Phonics 中点 | 1 | `0x00B7 ·` | 自然拼读音节分隔（"ap·ple"）|
| 中文标点 | 14 | CJK 标点 | `。，、；：！？""''（）《》` |
| 中文字符 | ~3000 | GB13000 一级字 | 小学常用字 + LLM 高频字 union |
| **合计** | **~3134** | | |

## 字体打包目标

| 字号 | 字符集 | 位深 | 用途 | Flash 估算 |

## 中文 3000 字来源（按权威性排序）

1. **《通用规范汉字表》一级字 3500 字** — 教育部 2013 年发布，覆盖小学 + 初中常用 [GB13000.1]
2. **《义务教育语文课程常用字表（一）》300 字** — 1-2 年级核心识字
3. **LLM tokenizer 高频字** — 仓库已有 `gen_ttf.py` 提取 DeepSeek/Qwen 词表 union

**实际取值**：以上三表交集 + 启蒙补充词（动物/植物/身体/家庭/食物名词），目标 ~3000 字。

**生成方式**：见 `components/78__xiaozhi-fonts/scripts/gen_edu_fonts.py`（A 任务交付脚本）。

## 拼音声调字符表（手列）

```
基础元音：a e i o u ü
带声调：
  ā á ǎ à
  ē é ě è
  ī í ǐ ì
  ō ó ǒ ò
  ū ú ǔ ù
  ǖ ǘ ǚ ǜ
辅音：b p m f d t n l g k h j q x zh ch sh r z c s y w
```

ü 拼音中两形：u 上加两点（ü）和带声调（ǖ ǘ ǚ ǜ）。

## Phonics 音节分隔字符

仅一个：**`·`** (U+00B7 MIDDLE DOT) — 用于 "ap·ple" / "el·e·phant" 等音节切分。

数据契约固定：左音节 + 中点 + 右音节，**不要用** `-` / `_` / 空格 等其他分隔符（避免与英文连字符混淆）。

## 不要包含的字符

- ❌ IPA 音标符号（ə ʃ θ ʒ æ ɪ 等）— v8 用 Phonics 替代 IPA
- ❌ 繁体字（除非简繁同形）— 启蒙用简体
- ❌ 日韩特有字符 — 中文产品不需要
- ❌ 数学符号（∑ ∫ √ 等）— 启蒙不涉及
- ❌ Emoji — 由独立 emoji 字体提供（已有 `font_emoji_32/64.c`）

## 字体生成命令

见 `scripts/gen_edu_fonts.py`：

```bash
# 一键生成 56 + 48 + 30 三档字体
cd components/78__xiaozhi-fonts
python3 scripts/gen_edu_fonts.py \
  --ttf ttf/975MaruSC-Regular.ttf \
  --output cbin/

# 输出：
#   cbin/font_maru_56_1.bin  (~1.2 MB · 1bpp 主秀)
#   cbin/font_maru_eng_48_4.bin  (~30 KB · 4bpp 英文兜底)
#   cbin/font_maru_30_4.bin   (~6 KB · 4bpp 顶部)
```

生成后在 `components/78__xiaozhi-fonts/include/cbin_font.h` 注册三个新字体名，UiDisplay 端 `EnsureDisplayFonts()` 加载。

## 工程视角关联判断

- **E1/E2/E3/G1 是一根藤**：息屏不进深睡 → 后台反复切网/联网 → 大声播报+靠近自动唤醒开机 → 同时拖垮续航。与 memory `post-prod-stability-2026-06`（弱网主任务、过放）方向一致，建议当一个**根因簇**统一查。
- **C1**（音量切网重置）与 G1 同属"切网副作用"，可能同一处状态被覆盖。
- **B1/B3** 偏端侧播放链路，**B2** 偏后端曲库/版权配置，需分开定位。
- **A1 新机闪红屏**是唯一明确硬件级，独立走换货 + 来料屏批次排查。

## 追加：bug/2.0版本/ 三视频分析（内部多台对比测试）

> 这三段是内部测试录像，逐台对比同款设备，定位价值高。

### 语音切换.mp4（165s）
- **D5 音色切换失败**：连喊"切换歪歪小盒"，设备答非所问（"你想聊点啥"），切不过去；测试者怀疑设备/智能体侧问题。
- **E3 强化**：测试中途弹"电量不足请充电"，再次印证掉电快。
- **C3 多台音量不一致**：同样设到 60%，几台实际音量大小不同。
- 有的机说话会卡、有的不卡（个体差异）。

### 按键音量，对话电流声.mp4（308s）
- **C1 精确根因**：音量 **≤40% 时，切换后会被重置成出厂/系统默认**；**≥50% 则不会**。即低音量设置不被持久化。
- **B4 电流声确认**：多台横评，电流声程度因机而异（有的明显、有的干净）——偏**硬件/批次**而非纯软件。
- **C3**：三台都调 60%，仍有一台明显偏小。
- **A4 音量调节动画时有时无**（"有时能调出来有时调不出"）。
- **A2 关联**：4G 状态下出现闪屏（"四级状态下有点闪屏，刚刚还好的"）。

### b3abb6c42033426c706f19e7fb859e2c.mp4（199s）
- **E4 开关机按键时长异常/疑似反了**：开机要长按 5~6 秒、关机只要 3~4 秒，测试者称"开关机键好像变反了，开机时间变长、关机时间变短"。
- 教学/识别响应不稳定（"花朵怎么读"有时能触发有时不能）。
- 电流声、卡顿因机而异（个体差异）。
